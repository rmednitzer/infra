# Security Policy

The security policy for this repository lives at
[`.github/SECURITY.md`](.github/SECURITY.md). GitHub renders that file as
the project security policy; this root-level stub exists so external
tooling (OpenSSF Scorecard, supply-chain scanners) that looks at the
repository root also finds it.

Report vulnerabilities through
[GitHub private vulnerability reporting](https://github.com/rmednitzer/infra/security/advisories/new).
